---------------- CONNECT TO PrimeIntellect Instance  (running pytorch/pytorch:2.4.0-cuda12.4.1-cudnn8-runtime) ----------------------
generate ssh certificate:
local> ssh-keygen -t ed25519 -f C:\Users\YOURUSERNAME\.ssh\primeintellect_ed25519 -C "yourPublicKeyName"
this will create C:\Users\YOURUSERNAME\.ssh\primeintellect_ed25519 (private key file) and C:\Users\YOURUSERNAME\.ssh\primeintellect_ed25519.pub (public key file)
put public key for instance connetion, then:
local> ssh root@IP -p THEPORT -i C:\Users\YOURUSERNAME\.ssh\primeintellect_ed25519
> whoami
> hostname
> pwd
> ls -la
> git clone https://github.com/radov02/Waymo-GNN.git
> cd Waymo-GNN
> apt-get update
> apt-get install -y ca-certificates curl gnupg lsb-release apt-transport-https software-properties-common

------------------------ GRAPH CREATION ---------------------------
> git pull
> chmod +x setup_remote.sh
> ./setup_remote.sh
> export WANDB_API_KEY='your-key-here'
> apt-get update
> apt-get install -y apt-transport-https ca-certificates gnupg curl
> curl -fsSL https://packages.cloud.google.com/apt/doc/apt-key.gpg | gpg --dearmor -o /usr/share/keyrings/cloud.google.gpg

> echo "deb [signed-by=/usr/share/keyrings/cloud.google.gpg] http://packages.cloud.google.com/apt cloud-sdk main" | tee /etc/apt/sources.list.d/google-cloud-sdk.list
> apt-get update
> apt-get install -y google-cloud-sdk
> gcloud init
> gcloud auth application-default login
(make sure billing account is set for the downloader project on google cloud)
> python src/graph_creation_and_saving.py

------------------------ TRAINING ---------------------------
(open more terminals using> ssh root@IP -p PORT -i C:\Users\YOURUSERNAME\.ssh\primeintellect_ed25519)
ter1> python src/autoregressive_predictions/training_batched.py
ter2> watch -n 1 nvidia-smi
(or better with graph:
ter2> apt-get update && apt-get install -y nvtop
